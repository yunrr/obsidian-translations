# 社区插件与主题 AI 汉化指南

请先确认译文版本，再下载对应文件。

## 译文版本

下载页面会显示译文版本号和插件版本号。一般先看译文版本，再看插件版本。

| 译文版本号 | 翻译方式        | 质量说明                                              |
| ---------- | --------------- | ----------------------------------------------------- |
| `1.0.0`    | AI 自动提取翻译 | 未经人工校对，可能存在机翻感或翻译不全的问题          |
| `1.0.1`    | AI 自动提取翻译 | 提取范围可能比 `1.0.0` 更广，但也更不稳定，更容易出错 |
| `2.0.0`    | 人工校对        | 经过人工精校，翻译更完整，表达也更自然                |

### 如何选择合适的翻译版本？

推荐顺序：

1. 优先选择 `2.0.0`。
2. 如果没有 `2.0.0`，再考虑 `1.0.1` 或 `1.0.0`。
3. 同一译文版本下，选择对应插件版本较新的文件。

![latex suite image](https://github.com/yunrr/obsidian-translations/blob/main/images/2026_03_15.jpg)

以上图 **Latex Suite** 为例，推荐选择译文 `v2.0.0`、插件 `1.9.8` 的文件，而不是译文 `v1.0.0`、插件 `1.11.0` 的文件。

## 找不到特定插件的译文怎么办？

如果用插件名称搜索不到，请改用插件 ID 搜索。

插件 ID 是本地 `plugins` 文件夹中的插件目录名。例如 BRAT 插件的目录名是 `obsidian42-brat`，插件 ID 就是 `obsidian42-brat`。

## 关于后续维护

本仓库文件数量较多，索引文件较大。频繁改动可能导致索引损坏，因此没有重大问题时，不会再频繁调整。

如果发现翻译文件有问题，可以自行修改，并上传到自己的翻译仓库作为自用译文。
